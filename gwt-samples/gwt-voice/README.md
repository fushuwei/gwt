# Development

## Step 1. Install *gwt-jelement* to local Maven repository

``git clone https://github.com/gwt-jelement/gwt-jelement.git``

``mvn install``

## Step 2. Build and Run

``mvn clean gwt:generate-module gwt:devmode``
